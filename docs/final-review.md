## TL;DR

> **"강력한 컨셉과 비주얼. 디테일·반응형·콘텐츠 깊이를 다듬으면 SOTD 경쟁권."**

현실에서 출발해 이진 데이터 터널을 지나 소프트웨어 코어·반도체 칩·PCB 커리큘럼·커리어/수상·미래로 이어지는 **단일 내러티브 스크롤**은 대학 홍보 사이트로는 드물게 야심차고 몰입감이 높다. Black + Deep Green(#00C853) 팔레트와 Matrix/PCB 모티프의 일관성도 우수하다. 다만 **전환부의 미세한 단절, 모바일 반응형, 접근성, 성능, 실제 홍보 콘텐츠의 깊이**에서 감점 요인이 있다.

## 기준별 상세 평가

### 1. Visual Quality — 8.7
**강점**
- 순수 Black 배경 + Deep Emerald(#00C853) + Mint Accent(#66FF99)의 절제된 팔레트. 네온의 저렴함을 걷어내 프리미엄·NVIDIA 톤 확보.
- JetBrains Mono 기반 타이포가 "코드/터미널" 정체성을 일관되게 전달.
- 데이터 글리프 터널, 응결하는 코어, 살아있는 PCB 등 **시그니처 비주얼**이 분명함.

**감점 요인**
- 일부 씬이 전반적으로 어둡고 대비가 낮아(코어 응결 중반, 커리어 그래프) 정보 가독성이 떨어짐.
- 좌상단 실제 대학 로고가 래스터(1.3MB)라 32px에서 뭉개지고 잘려 보임 — 픽셀 단위 완성도를 보는 심사에서 눈에 띄는 흠.
- 모바일에서 Hero 타이틀이 뷰포트 가장자리에 빡빡하게 붙음.

### 2. Storytelling — 8.3
**강점**
- "현실 → 데이터 공간 → 코어 → 회로(커리큘럼) → 커리어/수상 → 미래"라는 메타포가 **학과의 본질(0과 1로 세상을 만든다)** 과 정확히 맞물림.
- 각 씬의 캡션이 챕터를 명확히 안내. 고등학생·비전공자가 "탐험"의 감각을 느끼도록 설계됨.

**감점 요인**
- 일부 경계(Computer→Tunnel, Tunnel→Core)에 미세한 단절/점프가 남아 "완전 무이음"의 환상을 깨뜨림.
- 전체 스크롤 길이가 매우 김(~14,000px). 중반에 페이싱이 늘어져 이탈 위험. 감정 곡선의 "쉼표"와 "정점"이 더 또렷해야 함.

### 3. Interaction — 7.6
**강점**
- 스크롤 연동 터널·PCB 생성·코어 응결 등 **스크롤이 곧 카메라**인 설계.
- 커리어 노드 클릭 → 스택/프로젝트/수상/진로 드릴다운은 정보 탐색의 재미를 줌.
- 네비게이션 scroll-spy로 현재 위치를 정확히 표시.

**감점 요인**
- **접근성**: `prefers-reduced-motion` 미대응(터널·스파크·플래시는 광과민 위험). 키보드 포커스/ARIA 부재.
- 커스텀 커서(`cursor:none`)가 일부 환경에서 "커서 실종"으로 느껴질 수 있음.
- 클릭 가능 요소의 어포던스(hover hint)가 약함 — 사용자가 무엇을 누를 수 있는지 학습 비용 발생.
- 전역 진행도/챕터 인디케이터가 없어 긴 스크롤에서 길을 잃기 쉬움.

### 4. Branding — 6.6
**강점**
- SW·PCB·Matrix라는 **학문적 정체성의 시각화**가 일관됨. "SKU·SW CORE" 칩, 회로 커리큘럼 등 모티프가 메시지를 강화.
- 모노 타이포 + 그린 시스템으로 톤이 통일.

**감점 요인**
- **대학 공식 브랜딩이 약함**: 서경대 고유 색/심볼/서체 활용이 거의 없고, 좌상단 로고는 저해상 래스터로 품질 저하.
- 디지털-잎사귀 컨셉 로고(초기 제안)가 실제 래스터 로고로 대체되며 브랜드 일관성이 흐트러짐.
- 파비콘/메타(OG 이미지·타이틀)·소셜 공유 카드 등 브랜드 표면이 미정비.

### 5. University Promotion — 6.4
**강점**
- 커리큘럼(6트랙)·진로 5직군·수상·통계를 몰입형으로 제시. "와 멋있다 → 흥미롭다"의 1차 목표는 충분히 달성.
- CTA(입학 안내/커리큘럼 보기) 배치로 전환 동선 의도가 있음.

**감점 요인**
- **CTA가 死링크(`href="#"`)** — 홍보 사이트의 핵심 전환이 동작하지 않음(콘텐츠/사용성 동시 감점).
- 실제 정보 깊이 부족: 교수진·연구실·캠퍼스·입학 일정·실제 취업 데이터·재학생 인터뷰 부재. 수치가 다소 일반적.
- 국제 지원자/다국어(EN) 미지원. 검색·SEO·공유 최적화 미흡.

## 심사 총평

이 사이트는 **"학과를 설명하지 않고 소프트웨어 세계를 경험하게 한다"** 는 기획 의도를 시각·인터랙션 양면에서 상당 수준 구현했다. 컨셉의 독창성과 비주얼 완성도는 이미 Awwwards 후보권이다.

그러나 심사위원은 **마지막 10%의 디테일** — 무이음 전환, 모바일, 접근성, 살아있는 CTA, 브랜드 표면 — 에서 점수를 가른다. 위 Top 10 중 상위 5개(특히 #1~#5)를 정리하면, 현재 7.7 수준에서 **8.3~8.5(Site of the Day 경쟁권)** 로 올라설 수 있다.

**한 문장 평결**:
> "기술과 내러티브는 이미 수상권. 이제 '제품'으로서의 마감(반응형·접근성·전환·브랜드)을 끝내라."
